# gal
Get all possible href | src | url from target url or domain

### Dependency 
```
Linux System
curl
```

### Installation
```
git clone https://github.com/YashGoti/gal.git
cd gal
chmod +x gal
```
```
wget https://raw.githubusercontent.com/YashGoti/gal/master/gal && chmod +x gal
```

### Usage
```
$ ~/gal URL/DOMAIN
$ ~/gal URL/DOMAIN | grep '\.js$'
```
